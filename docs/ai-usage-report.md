# AI Usage Report – Personal Portfolio

## Tools Used
- **ChatGPT** – Assisted with code generation, bug fixing, and explanations.  
- **GitHub Copilot** – Provided auto-completion and function suggestions.

## Use Cases

- **Dark mode fix:** Used ChatGPT to adjust CSS variables for better readability. Edited colors, borders, shadows, text, and buttons to improve contrast.  
- **Project sorting and filtering:** Used ChatGPT to create JavaScript logic for filtering and sorting projects. Modified DOM manipulation code to ensure accessibility.  
- **Contact form validation:** Used GitHub Copilot to generate initial validation code. Added ARIA alerts and styled error messages for better user experience.  
- **Greeting and session timer:** Used ChatGPT to implement personalized greetings and a live session timer. Edited prompt logic and integrated localStorage for persistence.  
- **GitHub API integration:** Used ChatGPT to fetch the latest repository data and display it dynamically. Tested the API call and formatted the output for clarity.

## Benefits
- Accelerated prototyping and development.  
- Improved readability and accessibility, especially for dark mode.  
- Reduced time spent writing repetitive JavaScript logic.  
- Enhanced user experience through proper error handling and dynamic content.

## Challenges
- Adjusting dark mode colors to ensure sufficient contrast.  
- Ensuring accessibility compliance across dynamic elements.  
- Validating API calls and handling errors gracefully.

## Learning Outcomes
- Learned to integrate AI outputs responsibly while maintaining personal understanding.  
- Gained experience in adapting AI suggestions to fit project requirements.  
- Improved skills in dark mode design, accessibility, and user-focused front-end development.
